# linkbio

> 🖥️ Мой персональный сайт-визитка в стиле терминала

Нажми на ссылки, чтобы познакомиться со мной поближе 👇

## 🔗 Ссылки

В проекте собраны все мои соцсети и контакты в одном месте — как Linktree, только в формате терминала /about.exe.

## 🛠 Стек

- HTML + CSS + JS — чистый vanilla
- Стилизация под терминал Windows
- Деплой на [Vercel](https://vercel.com)

## 🚀 Запуск локально

```bash
# Просто открой index.html в браузере
# Или запусти локальный сервер:
npx serve .
```

## 📦 Деплой

Автоматически деплоится на Vercel через GitHub.
